# SemVer Tag Action

This action provides next semver tag (prerelease or patch).

## Examples

Next prerelease version:
```yaml
      - uses: Drill4J/semver-tag-action@v0.1.0
          with:
            type: prerelease
```

Next patch version:
```yaml
      - uses: Drill4J/semver-tag-action@v0.1.0
          with:
            type: prerelease
```
